## 🌐 Vercel 部署问题

### 问题描述
- Vercel 部署持续失败 (10+ 次 Error)
- 错误原因：`No Next.js version detected`
- 根本原因：Next.js 代码在 `frontend` 子目录，Vercel 默认在根目录构建

### 解决方案
**Build 设置需要配置：**
- **Root Directory**: `frontend`
- **Build Command**: `cd frontend && npm run build`
- **Output Directory**: `frontend/.next`
- **Install Command**: `cd frontend && npm install`

### 当前状态
- ✅ Settings 已配置正确的 Root Directory
- ❌ Redeploy 失败：`sh: line 1: cd: frontend: No such file or directory`
