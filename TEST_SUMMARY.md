## 三、测试环境配置详情

### 3.1 依赖包
- Playwright (@playwright/test) - E2E测试框架
- Jest (jest, @types/jest) - 单元测试框架
- TypeScript相关类型定义

### 3.2 测试目录结构
```
tests/
├── e2e/                    # 端到端测试
│   └── mood-checker.e2e.spec.ts
├── unit/                   # 单元测试 (待完善)
└── api/                    # API测试 (待完善)
```

### 3.3 测试脚本
- `npm run test` - 运行Jest单元测试
- `npm run test:e2e` - 运行Playwright E2E测试
- `npm run test:unit` - 运行单元测试
- `npm run test:api` - 运行API测试
- `npm run test:debug` - 调试模式运行Playwright测试
