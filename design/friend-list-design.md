# 好友列表设计规范

## 📋 项目概述

好友列表页面用于管理用户的好友关系，支持好友列表展示、在线状态显示、添加好友、好友请求处理等功能。

---

## 🎨 设计语言

### 主色调
- **主色**: `linear-gradient(135deg, #667eea, #764ba2)` - 紫蓝渐变
- **辅助色**: `#ffecd2, #fcb69f` - 暖橙渐变（好友请求）
- **成功色**: `#43e97b` - 绿色（在线状态）
- **离线色**: `#ccc` - 灰色（离线状态）
- **背景色**: `#fafafa` / `#f5f6fa` - 浅灰背景

### 字体
- **字体族**: `-apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif`
- **大标题**: 24px / 600
- **标题**: 20px / 600
- **正文**: 15px / 400
- **辅助文字**: 13px / 400
- **小组文字**: 13px / 600 (大写)

### 圆角
- **大卡片**: 20px
- **中等元素**: 16px
- **头像**: 14px - 24px
- **小按钮**: 10px
- **徽章**: 20px (pill shape)

### 阴影
- **卡片阴影**: `0 2px 8px rgba(0, 0, 0, 0.05)`
- **头像阴影**: `0 2px 8px rgba(102, 126, 234, 0.3)`
- **悬浮阴影**: `0 4px 12px rgba(102, 126, 234, 0.4)`
- **大阴影**: `0 20px 60px rgba(0, 0, 0, 0.3)`

## 🎭 交互规范

### 好友列表
1. **点击**: 选中好友，右侧/新页显示详情
2. **悬浮 (PC)**: 显示快捷操作 (发消息/语音)
3. **点击头像**: 打开完整个人资料
4. **下拉刷新**: 更新好友在线状态

### 好友请求
1. **点击请求卡片**: 打开请求列表弹窗
2. **接受请求**: 添加好友，发送欢迎消息
3. **拒绝请求**: 移除请求，可拉黑
4. **批量操作**: 全选接受/拒绝

### 添加好友
1. **搜索**: 支持用户名/ID/手机号
2. **搜索结果**: 显示匹配用户列表
3. **发送请求**: 填写验证消息，发送
4. **扫码添加**: 调用摄像头扫描二维码

### 好友详情
1. **发消息**: 跳转到聊天页面
2. **语音通话**: 发起语音呼叫
3. **视频通话**: 发起视频呼叫
4. **更多操作**: 备注/置顶/拉黑/删除

## ♿ 无障碍设计

### 颜色对比度
- 主要文字：`#1a1a1a` on `#ffffff` (对比度 16:1)
- 次要文字：`#666` on `#ffffff` (对比度 5:1)
- 辅助文字：`#999` on `#ffffff` (对比度 3:1)
- 在线状态：绿色需满足 WCAG AA 标准

### 键盘导航
- Tab 键顺序：搜索框 → 请求入口 → 筛选标签 → 好友列表 → 操作按钮
- Enter 键：选中好友/打开详情
- Escape 键：关闭弹窗/详情页
- 方向键：在好友列表中上下导航

### 屏幕阅读器
- 在线状态：`aria-label="在线"` / `aria-label="离线"`
- 未读请求：`aria-label="3 条好友请求"`
- 操作按钮：`aria-label="给大伟发消息"`

## 🔧 开发建议

### 性能优化
1. 虚拟滚动：好友列表超过 100 人时启用
2. 头像懒加载：使用 Intersection Observer
3. 状态轮询：在线状态每 30 秒更新一次
4. 防抖搜索：搜索输入防抖 300ms

### 数据加载
1. 分页加载：每页 50 个好友
2. 下拉刷新：更新在线状态
3. 无限滚动：滚动到底部自动加载
4. 预加载：详情数据预加载

### 本地存储
1. 好友列表：IndexedDB 缓存
2. 在线状态：localStorage + 定时刷新
3. 搜索历史：localStorage 存储最近 10 条
4. 分组展开状态：localStorage 持久化

### API 设计建议
```javascript
// 获取好友列表
GET /api/friends?page=1&limit=50&status=online

// 搜索好友
GET /api/friends/search?q=keyword

// 发送好友请求
POST /api/friend-requests { userId, message }

// 处理好友请求
PATCH /api/friend-requests/:id { action: 'accept' | 'reject' }

// 更新好友备注
PATCH /api/friends/:id { remark: '新备注' }

// 删除好友
DELETE /api/friends/:id
```

## 🎯 特殊场景

### 空状态
- **无好友**: 显示引导添加好友
- **无搜索结果**: 显示"未找到相关用户"
- **无好友请求**: 显示"暂无新请求"

### 加载状态
- **骨架屏**: 列表加载时显示骨架屏
- **头像占位**: 加载失败显示默认头像
- **下拉刷新**: 显示刷新动画

### 错误状态
- **网络错误**: 显示重试按钮
- **加载失败**: 显示错误提示
- **操作失败**: Toast 提示 + 原因
