## 二、心情颜色系统（7 种）

### 2.1 心情颜色定义

| ID | 心情 | Emoji | 颜色 | 渐变 | 使用场景 |
|----|------|-------|------|------|----------|
| 1 | 😄 开心 | 😄 | `#FFD700` | `linear-gradient(135deg, #FFD700 0%, #FFEC8B 100%)` | 积极情绪、高亮、庆祝 |
| 2 | 😊 平静 | 😊 | `#90EE90` | `linear-gradient(135deg, #90EE90 0%, #C1FFC1 100%)` | 平和状态、成功、稳定 |
| 3 | 😔 低落 | 😔 | `#87CEEB` | `linear-gradient(135deg, #87CEEB 0%, #B0E0F6 100%)` | 消极情绪、提醒、关注 |
| 4 | 😠 生气 | 😠 | `#FF6B6B` | `linear-gradient(135deg, #FF6B6B 0%, #FF8E8E 100%)` | 警告、强烈情绪、急需关注 |
| 5 | 😰 焦虑 | 😰 | `#DDA0DD` | `linear-gradient(135deg, #DDA0DD 0%, #E8C4E8 100%)` | 关注、需要帮助、压力 |
| 6 | 😴 疲惫 | 😴 | `#D3D3D3` | `linear-gradient(135deg, #D3D3D3 0%, #E8E8E8 100%)` | 中性、未打卡、休息 |
| 7 | 🤩 兴奋 | 🤩 | `#FF69B4` | `linear-gradient(135deg, #FF69B4 0%, #FF8ECA 100%)` | 特殊时刻、庆祝、高光 |

### 2.2 心情颜色应用规范

#### 心情选择器
- 未选中：纯色背景 + emoji
- 选中：渐变背景 + 白色描边 + 阴影
- 悬停：亮度提升 10%

#### 日历渲染
- 已打卡：心情颜色填充日期圆圈
- 未打卡：`#F3F4F6` 浅灰色
- 今日：蓝色描边 `#6366F1`

#### 统计图表
- 饼图/柱状图使用心情渐变色的 80% 不透明度

## 四、阴影规范

### 4.1 阴影层级

```css
/* 阴影 - 小 */
shadow-sm: 0 1px 2px 0 rgba(0, 0, 0, 0.05)

/* 阴影 - 中 */
shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px 0 rgba(0, 0, 0, 0.06)

/* 阴影 - 大 */
shadow-md: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)

/* 阴影 - 超大 */
shadow-lg: 0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)

/* 阴影 - 心情卡片专用（彩色阴影） */
shadow-mood: 0 4px 14px 0 rgba(99, 102, 241, 0.25)
```

## 五、圆角规范

### 5.1 圆角层级

```css
/* 圆角 - 小 */
rounded-sm: 2px

/* 圆角 - 中 */
rounded: 4px

/* 圆角 - 大 */
rounded-md: 6px

/* 圆角 - 超大 */
rounded-lg: 8px

/* 圆角 - 卡片 */
rounded-xl: 12px

/* 圆角 - 按钮 */
rounded-full: 9999px
```

### 5.2 应用场景

| 元素 | 圆角 | 示例 |
|------|------|------|
| 按钮 | `rounded-lg` (8px) | 主要/次要按钮 |
| 输入框 | `rounded-md` (6px) | 文本输入、密码输入 |
| 卡片 | `rounded-xl` (12px) | 内容卡片、统计卡片 |
| 心情选择器 | `rounded-full` | 圆形心情按钮 |
| 标签 | `rounded-full` | 标签徽章 |
| 头像 | `rounded-full` | 用户头像 |
| 日历日期 | `rounded-full` | 日期圆圈 |
