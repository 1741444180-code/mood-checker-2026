## ✨ 交互效果

### 标签页切换

```css
    transition: all 0.3s ease;

    background: linear-gradient(135deg, #f39c12, #e74c3c);
}
```

## 🏆 排名视觉体系

### 前 3 名特殊处理
```
第 1 名：🥇 金色 + 大号头像 + 皇冠图标
第 2 名：🥈 银色 + 中号头像
第 3 名：🥉 铜色 + 中号头像
4 名及以后：普通样式
```

### 趋势标识
```
上升：↑ + 绿色 + 上升位数
下降：↓ + 红色 + 下降位数
持平：─ + 灰色
新入榜：NEW + 高亮
```

## 🔒 性能优化

### 渲染优化
- 使用 CSS 渐变代替图片
- 列表虚拟化 (长列表时)
- 适度使用 backdrop-filter
- 动画使用 transform 和 opacity

### 加载策略
- 首屏优先加载前 50 名
- 滚动加载更多
- 头像使用懒加载
- 骨架屏占位
