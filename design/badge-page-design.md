## 🎯 组件规范

### 徽章卡片
- **尺寸：** 280px × 280px (PC) / 160px × 220px (移动)
- **圆角：** 20px (PC) / 16px (移动)
- **背景：** rgba(255, 255, 255, 0.08)
- **边框：** 2px solid transparent
- **悬停效果：** translateY(-8px) + 边框高亮

### 徽章图标
- **尺寸：** 120px × 120px (PC) / 70px × 70px (移动)
- **形状：** 圆形
- **渐变：** #f39c12 → #e74c3c → #9b59b6
- **锁定状态：** grayscale(100%) + opacity(0.5)

### 进度条
- **高度：** 8px (PC) / 6px (移动)
- **背景：** rgba(255, 255, 255, 0.1)
- **填充：** linear-gradient(90deg, #f39c12, #e74c3c)
- **圆角：** 4px (PC) / 3px (移动)

### 统计栏
- **背景：** rgba(255, 255, 255, 0.05)
- **圆角：** 20px (PC) / 16px (移动)
- **内边距：** 30px (PC) / 20px 16px (移动)
- **模糊效果：** backdrop-filter: blur(10px)

## ✨ 交互效果

### 悬停效果
```css
.badge-card:hover {
    transform: translateY(-8px);
    background: rgba(255, 255, 255, 0.12);
    border-color: rgba(243, 156, 18, 0.3);
    box-shadow: 0 20px 40px rgba(0, 0, 0, 0.3);
}
```

### 点击效果 (移动端)
```css
.badge-card:active {
    transform: scale(0.96);
}
```

### 模态框动画
```css
@keyframes slideUp {
    from { transform: translateY(100%); }
    to { transform: translateY(0); }
}
```

## 🏆 徽章等级体系

### 稀有度分级
1. **普通 (Common)** - 灰色，基础成就
2. **稀有 (Rare)** - 蓝色，中等难度
3. **史诗 (Epic)** - 紫色，高难度
4. **传说 (Legendary)** - 金色，极高难度

### 解锁状态
- **已解锁：** 彩色图标 + 勾选标记 + 完整进度
- **未解锁：** 灰度图标 + 半透明 + 进度显示

- 动画使用 transform 和 opacity

- 支持键盘导航
- 触摸目标尺寸 ≥ 44px
