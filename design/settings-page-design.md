## 🧩 组件规范

### 1. 开关 Toggle

**尺寸：** 50px × 28px (PC) / 48px × 26px (Mobile)  
**圆角：** 14px (PC) / 13px (Mobile)  
**滑块：** 20px × 20px

**状态：**
- 关闭：背景 #e0e0e0
- 开启：渐变主色 #667eea → #764ba2

**交互：**
- 点击切换状态
- 过渡动画：0.3s ease

**主要按钮：**

- 圆角：8px (PC) / 12px (Mobile)
- 悬停：上移 2px + 阴影

**次要按钮：**

- 文字：#333
- 悬停：背景 #e0e0e0

**危险按钮：**
- 背景：#ff4757
- 文字：白色
- 悬停：背景 #ff3344

**尺寸：**

## 📄 页面内容规范

### 个人信息模块

**字段：**
- 头像（可更换）
- 昵称（文本输入）
- 邮箱（邮箱输入）
- 手机号（电话输入）
- 生日（日期选择）

**操作：**
- 保存修改（主要按钮）
- 取消（次要按钮）

### 账号安全模块

**字段：**
- 当前密码（密码输入）
- 新密码（密码输入 + 规则提示）
- 确认新密码（密码输入）

**开关：**
- 两步验证（默认关闭）
- 登录提醒（默认开启）

**操作：**
- 更新密码（主要按钮）
- 查看登录记录（次要按钮）

### 通知偏好模块

**开关：**
- 系统通知（默认开启）
- 活动提醒（默认开启）
- 邮件通知（默认关闭）
- 短信通知（默认开启）

**免打扰时段选择：**
- 夜间模式（22:00 - 08:00）
- 自定义时段
- 完全静音

### 隐私管理模块

**快捷入口：**
- 数据可见性
- 隐私模式
- 数据导出
- 账号注销

**操作：**
- 管理隐私（主要按钮，跳转隐私页）
- 注销账号（危险按钮）

**警告提示：**
- 背景：#fff3cd
- 边框：4px solid #ffc107
- 文字：#856404

## 🔐 隐私页面规范

### 数据可见性

**选项：**
- 公开个人资料（默认开启）
- 显示在线状态（默认开启）
- 公开游戏成就（默认关闭）
- 显示游戏动态（默认开启）

### 数据导出

**数据类型：**
- 个人信息（约 2 MB）
- 游戏记录（约 15 MB）
- 设置数据（约 0.5 MB）

**导出格式：**
- JSON（结构化，默认）
- CSV（表格格式）
- PDF（可读报告）

**流程：**
1. 选择数据类型
2. 选择导出格式
3. 提交申请
4. 等待处理（24 小时内）
5. 下载加密压缩包

### 账号注销

**警告信息：**
- 所有数据永久删除
- 游戏进度清空
- 已购服务终止
- 用户名释放

**注销流程：**
1. 确认了解后果，备份数据
2. 取消订阅和自动续费
3. 验证身份（短信/邮箱）
4. 等待 7 天冷静期
5. 7 天后永久删除

**确认机制：**
- 必须勾选确认框
- 按钮初始禁用，勾选后启用

## ✨ 交互规范

### 过渡动画
```
所有交互元素：transition: all 0.3s
开关切换：transition: all 0.3s
按钮悬停：transform: translateY(-2px)
进度条：transition: width 0.3s
```

### 交互反馈
- 按钮点击：轻微缩放效果
- 开关切换：平滑滑动动画
- 输入框聚焦：边框变色 + 阴影
- 选项卡切换：背景色变化

### 表单验证
- 必填项：红色边框 + 错误提示
- 密码强度：实时显示强度条
- 邮箱格式：失焦时验证
- 手机号格式：失焦时验证

## 📊 响应式断点

```
移动端：≤ 480px
平板端：481px - 768px
PC 端：≥ 769px
```

**适配策略：**
- 移动端：单列布局，卡片垂直堆叠
- PC 端：双列网格布局，卡片并排显示
- 字体大小、间距、圆角根据平台调整

## 🎯 设计原则

1. **清晰直观：** 信息层级分明，用户一眼找到所需功能
2. **操作简便：** 常用操作触手可及，减少点击次数
3. **视觉统一：** 遵循统一的设计系统，保持品牌一致性
4. **安全提示：** 敏感操作（如注销）提供充分警告
5. **响应式设计：** PC 和移动端都有良好体验
