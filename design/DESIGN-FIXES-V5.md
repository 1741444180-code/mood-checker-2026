## ✅ 需求说明

当弹窗界面出现时：
1. ✅ 弹窗界面保持正常亮度
2. ✅ 原来的背景窗口要变暗（添加半透明黑色遮罩）
3. ✅ 视觉上有主次感觉
4. ✅ 用户注意力会放在弹窗上
5. ✅ 不会混淆按钮和界面

---

## 🎨 技术实现

### CSS 代码

```css
/* 背景遮罩层 */
.modal-overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0, 0, 0, 0.5); /* 50% 透明度黑色 */
  backdrop-filter: blur(4px); /* 毛玻璃效果 */
  z-index: 1000;
}

/* 弹窗内容 */
.modal-content {
  position: fixed;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  background: white;
  z-index: 1001;
  box-shadow: 0 10px 40px rgba(0,0,0,0.3);
}
```

### 关键参数

| 属性 | 值 | 说明 |
|------|-----|------|
| 遮罩透明度 | `rgba(0, 0, 0, 0.5)` | 50% 黑色，足够暗但不完全遮挡 |
| 毛玻璃模糊 | `blur(4px)` | 轻微模糊，增强层次感 |
| 遮罩 z-index | `1000` | 覆盖背景 |
| 弹窗 z-index | `1001` | 覆盖遮罩 |
| 弹窗阴影 | `0 10px 40px rgba(0,0,0,0.3)` | 深度感，突出弹窗 |
