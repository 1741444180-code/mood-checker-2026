## 四、前端开发对接

### 4.1 依赖安装

```bash
# 图标库
npm install lucide-react

# 动画库（可选）
npm install framer-motion
```

### 4.2 Tailwind 配置

在 `tailwind.config.js` 中添加：

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: '#667eea',
          hover: '#5568d3',
          light: '#E0E7FF',
        },
        mood: {
          happy: '#FFD700',
          calm: '#90EE90',
          sad: '#87CEEB',
          angry: '#FF6B6B',
          anxious: '#DDA0DD',
          tired: '#D3D3D3',
          excited: '#FF69B4',
        },
      },
      boxShadow: {
        'mood': '0 4px 14px 0 rgba(99, 102, 241, 0.25)',
      },
    },
  },
}
```
