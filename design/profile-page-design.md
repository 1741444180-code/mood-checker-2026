
## 📐 布局规范

### PC 端布局

- **容器宽度**: max-width: 1200px
- **网格结构**: 380px (左侧) + 1fr (右侧)
- **间距**: gap: 30px
- **页面内边距**: padding: 40px

### 移动端布局

- **容器宽度**: max-width: 600px
- **单列流式布局**
- **卡片外边距**: margin: 0 15px
- **底部导航**: fixed + safe-area-inset-bottom

## 🧩 组件规范

### 1. 用户信息卡片

**PC 端**:
- 头像尺寸：120px × 120px
- 边框：5px solid rgba(255,255,255,0.4)
- 用户名：1.8rem / 700 字重
- ID 标签：背景 rgba(255,255,255,0.2) + 圆角 20px

**移动端**:
- 头像尺寸：90px × 90px
- 边框：4px solid rgba(255,255,255,0.5)
- 用户名：1.5rem / 700 字重
- ID 标签：padding 5px 14px

### 2. 统计卡片

**三列布局**:
- 打卡天数
- 心情徽章
- 连续打卡

**数值样式**:
- 字体大小：2rem - 2.2rem
- 字重：700
- 渐变文字：linear-gradient(#667eea, #764ba2)

### 3. 菜单列表项

**结构**:
```
[图标 45-50px] [标题 + 描述] [箭头/Badge]
```

**图标背景渐变**:
- 蓝色：linear-gradient(#e3f2fd, #bbdefb)
- 绿色：linear-gradient(#e8f5e9, #c8e6c9)
- 橙色：linear-gradient(#fff3e0, #ffe0b2)
- 紫色：linear-gradient(#f3e5f5, #e1bee7)
- 红色：linear-gradient(#ffebee, #ffcdd2)
- 灰色：linear-gradient(#f5f5f5, #e0e0e0)

**交互效果**:
- PC: hover 时背景渐变 + 箭头右移 5px
- 移动：active 时背景 #f8f9fa + 箭头右移

### 4. 开关组件 (Toggle)

**尺寸**:
- PC: 56px × 32px
- 移动：52px × 30px

**状态**:
- ON: 渐变背景 (#667eea → #764ba2)
- OFF: 灰色背景 #e0e0e0

**滑块**:
- 尺寸：24-26px 圆形
- 阴影：0 2px 8px rgba(0,0,0,0.15)
- 过渡：transform 0.3s

### 5. 按钮

**主要按钮**:
- 背景：渐变 (#667eea → #764ba2)
- 文字：white
- 阴影：0 4px 15px rgba(102, 126, 234, 0.3)
- 悬停：translateY(-3px) + 增强阴影

**危险按钮 (退出登录)**:
- 背景：white
- 边框：2px solid #ff4444
- 文字：#ff4444
- 悬停：背景变红 + 文字变白

### 6. 徽章展示

**PC 端**: 4 列网格
**移动端**: 横向滚动

**单个徽章**:
- 尺寸：70px (移动) / auto (PC)
- 背景：渐变 (#f8f9fa → #e8eaf6)
- 表情：2rem
- 名称：0.7-0.8rem

**交互**:
- 悬停/点击时背景变渐变 + 表情反色

### 7. 底部导航 (仅移动端)

**样式**:
- 固定底部
- 背景：white
- 边框：top 1px solid #e0e0e0
- 阴影：0 -4px 20px rgba(0,0,0,0.08)

**导航项**:
- 图标：1.6rem
- 文字：0.7rem
- 激活色：#667eea
- 默认色：#999

5. **交互动效** - 悬停/点击反馈一致
