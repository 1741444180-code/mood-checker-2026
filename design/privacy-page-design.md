## 📄 页面内容规范

### 模块一：数据可见性

**标题：** 📊 数据可见性  
**图标尺寸：** 36px × 36px

**选项列表：**
1. **公开个人资料**（默认开启）
   - 描述：其他用户可以查看您的昵称和头像
   
2. **显示在线状态**（默认开启）
   - 描述：其他用户可以看到您是否在线
   
3. **公开游戏成就**（默认关闭）
   - 描述：其他用户可以查看您的游戏成就和排名
   
4. **显示游戏动态**（默认开启）
   - 描述：其他用户可以看到您的游戏活动记录

### 模块二：数据导出

**标题：** 📥 数据导出

**说明框内容：**
```
💡 导出数据说明
您可以随时下载您的个人数据副本，包含您的账号信息、
游戏记录、设置偏好等。数据将以加密压缩包形式提供，
通常在 24 小时内准备完成。
```

**数据类型（3 选多）：**
- 👤 个人信息（约 2 MB）
- 🎮 游戏记录（约 15 MB）
- ⚙️ 设置数据（约 0.5 MB）

**导出格式（单选）：**
- 📋 JSON - 结构化数据，适合程序读取（默认）
- 📊 CSV - 表格格式，适合 Excel 打开
- 📄 PDF - 可读报告，适合打印

**操作按钮：**
- 申请导出数据（主要按钮）
- 查看导出历史（次要按钮）

**进度条信息：**
- 上次导出：2026-02-15
- 状态：已完成

### 模块三：账号注销

**标题：** 🗑️ 账号注销

**警告列表：**
```
⚠️ 注销前请三思
• 所有个人数据将被永久删除，无法恢复
• 游戏进度、成就、虚拟货币将全部清空
• 已购买的服务和订阅将立即终止
• 用户名将被释放，其他人可以注册
```

**注销流程（5 步）：**
1. 确认您已了解注销后果，并备份需要保留的数据
2. 取消所有活跃的订阅和自动续费服务
3. 验证您的身份（短信或邮箱验证码）
4. 等待 7 天冷静期，期间可随时撤销注销申请
5. 7 天后账号和数据将被永久删除

**确认机制：**
- 复选框文案：「我已阅读并理解账号注销的后果，确认要永久删除我的账号」
- 按钮初始状态：禁用
- 勾选后状态：启用

**操作按钮：**
- 申请注销账号（危险按钮，初始禁用）
- 联系客服（次要按钮）

**客服信息：**
```
如有任何问题或疑虑，请联系客服邮箱：
support@xiaoyagame.com
```

## 🔒 安全规范

### 敏感操作确认
1. **账号注销：** 必须勾选确认框
2. **二次验证：** 提交时可能需要短信/邮箱验证
3. **冷静期：** 注销申请后 7 天冷静期

### 数据保护
1. **导出加密：** 数据压缩包加密处理
2. **访问控制：** 仅用户本人可操作
3. **操作日志：** 所有敏感操作记录日志

### 提示充分性
1. **后果告知：** 注销前明确告知所有后果
2. **流程透明：** 清晰展示注销流程和时间
3. **客服支持：** 提供客服联系方式

## 🎯 设计原则

1. **信息清晰：** 敏感操作提供充分的信息和警告
2. **操作审慎：** 注销等危险操作需要多重确认
3. **视觉引导：** 使用颜色区分信息、警告、危险等级
4. **流程透明：** 清晰展示操作步骤和时间预期
5. **用户友好：** 提供客服支持，方便用户咨询
