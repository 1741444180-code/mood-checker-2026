## 🎨 设计目标

为用户提供一个直观、高效的数据导出界面，支持多种导出格式、灵活的日期范围选择和自定义字段配置，同时提供清晰的导出进度反馈。

## 🧩 组件规范

### 1. 导出格式选择器

**PC 端：** 3 列网格布局  
**移动端：** 垂直列表布局

**状态：**
- 默认：灰色边框 #e2e8f0
- 悬停：紫色边框 #667eea，轻微上移
- 选中：紫色边框 + 渐变背景 + 对勾标记

**尺寸：**
- PC: 每项约 280px 宽
- 移动：全宽，高度约 80px

### 2. 日期范围选择器

**输入框：**
- 边框：2px solid #e2e8f0
- 圆角：8px (PC) / 10px (移动)
- 聚焦：紫色边框 + 阴影

**快捷按钮：**
- 默认：白色背景，灰色边框
- 激活：渐变背景，白色文字
- 圆角：6px (PC) / 20px (移动，胶囊形)

### 3. 字段选择器

**复选框样式：**
- 尺寸：18px × 18px (PC) / 20px × 20px (移动)
- 颜色：紫色 #667eea
- 选中状态：紫色边框 + 渐变背景

**布局：**
- PC: 4 列网格
- 移动：单列垂直

### 4. 操作按钮

**主按钮（开始导出）：**
- 背景：渐变紫色
- 文字：白色
- 圆角：12px
- 尺寸：PC (16px 字重), 移动 (16px)
- 交互：上移 2px + 阴影

**次按钮（取消）：**
- 背景：白色 / #f7fafc
- 边框：#e2e8f0
- 文字：#4a5568

### 5. 进度弹窗

**遮罩层：**
- 背景：rgba(0, 0, 0, 0.6)
- 模糊：backdrop-filter: blur(4px)

**弹窗主体：**
- 背景：白色
- 圆角：20px
- 最大宽度：500px
- 内边距：40px (PC) / 32px 24px (移动)

**进度条：**
- 高度：12px (PC) / 10px (移动)
- 背景：#e2e8f0
- 填充：渐变紫色 + 闪烁动画

**动画：**
- 弹窗进入：slideUp 0.3s ease
- 进度条：shimmer 2s infinite
- 图标：pulse 2s infinite

### 6. Toast 提示

**位置：**
- PC: 右上角 fixed
- 移动：顶部居中 fixed

**样式：**
- 背景：白色
- 圆角：12px
- 阴影：0 10px 40px rgba(0, 0, 0, 0.15)
- 动画：slideIn / fadeIn 0.3s

## 🎭 交互规范

### 点击反馈

- **按钮：** scale(0.98) 或 translateY(-2px)
- **卡片选项：** 边框变色 + 背景渐变
- **复选框：** 即时切换状态

### 过渡动画

```css
transition: all 0.3s ease;
```

### 加载状态

- 进度条闪烁效果
- 实时数据更新（行数、大小、时间）
- 百分比数字动态变化

## 📊 响应式断点

```css
/* 移动端优先 */
@media (max-width: 768px) {
  /* 格式选择器：3 列 → 1 列 */
  /* 字段选择器：4 列 → 2 列 */
  /* 导出按钮：横向 → 纵向 */
  /* 进度弹窗：居中 → 底部弹出 */
}
```

## 📱 移动端优化

1. **底部固定操作栏** - 确保主要操作始终可见
2. **触摸优化** - 移除高亮，优化点击反馈
3. **横向滚动** - 快捷日期按钮支持横向滚动
4. **底部弹窗** - 进度弹窗在移动端从底部滑出
5. **统计卡片** - 顶部增加数据概览卡片

---

## 🔍 设计亮点

1. **渐变色彩系统** - 统一的紫色渐变主题，视觉一致性强
2. **卡片式布局** - 信息分组清晰，层次分明
3. **微交互动画** - 悬停、点击、加载都有细腻的反馈
4. **进度可视化** - 多维度展示导出进度（百分比、行数、大小、时间）
5. **响应式设计** - PC 和移动端有各自最优的布局方案
6. **快捷操作** - 日期快速选择、字段全选/清除
