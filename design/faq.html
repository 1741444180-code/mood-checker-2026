🎮 如何注册新账号？

注册账号非常简单：

1. 点击首页右上角的"注册"按钮

2. 填写您的邮箱地址和设置密码

3. 验证邮箱后即可完成注册

4. 也可以使用微信、QQ 等第三方账号快速登录

💰 充值后未到账怎么办？

如果遇到充值未到账的情况：

1. 首先检查支付是否成功（查看银行/支付平台扣款记录）

2. 等待 1-2 分钟，系统可能有延迟

3. 刷新页面或重新登录账号

4. 如仍未到账，请联系客服并提供订单号

🔐 忘记密码如何找回？

找回密码步骤：

1. 在登录页面点击"忘记密码"

2. 输入您注册时使用的邮箱

3. 查收邮件中的重置链接

4. 点击链接设置新密码

5. 使用新密码重新登录

🎯 游戏有哪些玩法模式？

我们提供多种游戏模式：

• 经典模式：传统玩法，适合新手

• 竞技模式：排位赛，赢取奖励

• 挑战模式：高难度关卡，测试技巧

• 娱乐模式：轻松休闲，无压力

📱 支持哪些设备和系统？

我们支持以下平台和设备：

• macOS 10.15 及以上

• iOS 13.0 及以上

• Android 8.0 及以上

• 网页版：Chrome、Safari、Edge 等主流浏览器

我们支持多种支付方式：

• 银联云闪付

🔄 如何申请退款？

退款申请流程：

1. 进入"我的订单"页面

2. 找到需要退款的订单

3. 点击"申请退款"按钮

4. 填写退款原因并提交

5. 客服将在 1-3 个工作日内处理
