# 消息中心设计规范

## 📋 项目概述

消息中心是用户查看和管理所有消息的核心页面，支持消息列表展示、未读标记、消息详情查看以及删除/已读等操作。

---

## 🎨 设计语言

### 主色调
- **主色**: `linear-gradient(135deg, #667eea, #764ba2)` - 紫蓝渐变
- **辅助色**: `#ff6b6b` - 红色（用于未读标记、危险操作）
- **成功色**: `#43e97b` - 绿色（在线状态）
- **背景色**: `#fafafa` / `#f5f6fa` - 浅灰背景

### 字体
- **字体族**: `-apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif`
- **标题**: 20px / 600
- **正文**: 15px / 400
- **辅助文字**: 13px / 400
- **时间戳**: 12px / 400

### 圆角
- **大卡片**: 16px
- **中等元素**: 12px
- **小按钮**: 8px
- **徽章**: 20px (pill shape)

### 阴影
- **卡片阴影**: `0 2px 8px rgba(0, 0, 0, 0.05)`
- **悬浮阴影**: `0 4px 12px rgba(102, 126, 234, 0.3)`
- **大阴影**: `0 20px 60px rgba(0, 0, 0, 0.3)`

## 🧩 组件规范

### 1. 消息列表项

**PC 端**:
- 尺寸：`width: 100%, padding: 16px`
- 头像：`48x48px, border-radius: 12px`
- 间距：`gap: 12px`
- 未读状态：左侧 3px 边框 + 渐变背景
- 选中状态：渐变背景 + 白色文字
- 悬浮效果：向右平移 4px + 边框高亮

**移动端**:
- 尺寸：`padding: 14px`
- 头像：`52x52px, border-radius: 14px`
- 未读标记：右侧 8px 红点
- 点击反馈：scale(0.98)
- 滑动操作：显示已读/删除按钮

### 2. 未读徽章
- 背景：`linear-gradient(135deg, #ff6b6b, #ee5a6f)`
- 文字：白色，12px, 600
- 内边距：`4px 10px`
- 圆角：12px
- 最小宽度：24px

### 3. 筛选标签
- 默认：灰色文字，透明背景
- 激活：渐变背景 + 白色文字
- 圆角：8px (PC) / 20px (移动)
- 内边距：`8px 16px`

### 4. 操作按钮

**主按钮**:
- 背景：渐变紫色
- 文字：白色
- 圆角：12px
- 悬浮：上移 2px + 阴影

**次要按钮**:
- 背景：`#f5f5f5`
- 文字：`#666`
- 悬浮：背景变深

**危险按钮**:
- 背景：`#fff5f5`
- 文字：`#ff6b6b`
- 悬浮：`#ffe3e3`

## 🎭 交互规范

### 消息列表
1. **点击**: 选中消息，右侧显示详情
2. **悬浮 (PC)**: 显示操作按钮，背景高亮
3. **滑动 (移动)**: 显示已读/删除操作
4. **长按 (移动)**: 弹出批量操作菜单

### 消息详情
1. **标记已读**: 点击已读按钮，移除未读标记
2. **删除消息**: 点击删除按钮，二次确认后删除
3. **置顶消息**: 点击图钉按钮，消息置顶
4. **静音通知**: 点击铃铛按钮，屏蔽通知

### 批量操作
1. **全部已读**: 一键标记所有消息为已读
2. **清空**: 删除所有消息，需二次确认

## ♿ 无障碍设计

### 颜色对比度
- 主要文字：`#1a1a1a` on `#ffffff` (对比度 16:1)
- 次要文字：`#666` on `#ffffff` (对比度 5:1)
- 辅助文字：`#999` on `#ffffff` (对比度 3:1)

### 键盘导航
- Tab 键顺序：搜索框 → 筛选标签 → 消息列表 → 操作按钮
- Enter 键：选中/打开消息
- Escape 键：关闭详情页/弹窗

### 屏幕阅读器
- 未读消息：添加 `aria-label="未读消息"`
- 操作按钮：添加 `aria-label="标记已读"` 等描述
- 动态更新：使用 `aria-live` 区域

## 🔧 开发建议

### 性能优化
1. 虚拟滚动：消息列表超过 50 条时启用
2. 图片懒加载：头像使用懒加载
3. 防抖搜索：搜索输入防抖 300ms

### 数据加载
1. 分页加载：每页 20 条消息
2. 下拉刷新：支持 Pull-to-Refresh
3. 无限滚动：滚动到底部自动加载

### 本地存储
1. 未读计数：localStorage 缓存
2. 最后阅读位置：sessionStorage 记录
3. 筛选状态：localStorage 持久化
