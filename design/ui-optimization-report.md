## 🔍 发现的问题

### 1. 色彩系统不一致 ⚠️ 严重

#### 问题描述
不同页面使用了多套主色调，缺乏统一的品牌色彩系统。

| 页面类型 | 使用的主色 | 问题 |
|----------|-----------|------|
| 通知设置、日历、设置页、隐私页 | `#667eea` → `#764ba2`（紫色渐变） | ✅ 主系统 |
| 成就徽章、排行榜 | `#1a1a2e` → `#16213e` → `#0f3460` + `#f39c12`（深色游戏风） | ⚠️ 风格偏离 |
| 数据报表、日志审计 | `#1a1a2e` → `#16213e` + `#4fc3f7`（深色 + 青色） | ⚠️ 风格偏离 |
| 后台管理系统 | `#667eea` → `#764ba2`（紫色渐变） | ✅ 主系统 |

#### 影响
- 品牌识别度降低
- 用户在不同页面间切换时产生割裂感
- 开发维护成本增加

#### 建议方案
**统一采用紫色渐变 `#667eea` → `#764ba2` 作为主品牌色**

特殊页面（如成就徽章、排行榜）可在保持主色基础上，通过辅助色（金色 `#f39c12`）来营造游戏氛围，但主框架应保持一致。

---

### 2. 容器宽度不统一 ⚠️ 中等

#### 问题描述
页面容器最大宽度存在多种规格。

| 页面 | 容器宽度 | 问题 |
|------|----------|------|
| 通知设置页 | `max-width: 900px` | ✅ 推荐 |
| 设置页/隐私页 | `max-width: 900px` (双列布局) | ✅ 推荐 |
| 日历视图页 | `max-width: 1200px` | ⚠️ 偏大 |
| 数据报表页 | 侧边栏 240px + 自适应 | ⚠️ 布局不同 |
| 成就徽章页 | `max-width: 1400px` | ⚠️ 过大 |

#### 建议方案
建立统一的容器宽度规范：

| 页面类型 | 容器宽度 | 说明 |
|----------|----------|------|
| 设置类页面 | `900px` | 单列/双列表单 |
| 仪表盘/报表 | `1200px` | 数据展示 |
| 游戏/成就类 | `1400px` | 沉浸式体验 |
| 移动端 | `100%` | 全屏适配 |

---

### 3. 圆角规范混乱 ⚠️ 中等

#### 问题统计

| 元素类型 | 发现的范围 | 推荐值 |
|----------|-----------|--------|
| 大卡片容器 | 16px - 24px | **20px** |
| 中等卡片 | 12px - 20px | **16px** |
| 按钮 | 8px - 16px | **12px** |
| 输入框 | 8px - 12px | **10px** |
| 徽章/Badge | 16px - 20px | **20px** (完全圆角) |
| 头像 | 50% | ✅ 统一 |

#### 建议方案
```css
/* 统一圆角规范 */
--radius-lg: 20px;    /* 大卡片、容器 */
--radius-md: 16px;    /* 中等卡片、弹窗 */
--radius-sm: 12px;    /* 按钮、输入框 */
--radius-xs: 8px;     /* 小元素、标签 */
--radius-full: 9999px; /* 完全圆角（徽章、开关） */
```

### 4. 字体层级不清晰 ⚠️ 中等

#### 问题描述
页面标题、章节标题、正文的字号缺乏统一层级。

| 元素 | 发现的范围 | 推荐值 |
|------|-----------|--------|
| 页面大标题 (H1) | 28px - 48px | **32px** (PC) / **24px** (Mobile) |
| 章节标题 (H2) | 20px - 28px | **24px** (PC) / **20px** (Mobile) |
| 卡片标题 (H3) | 16px - 20px | **18px** (PC) / **16px** (Mobile) |
| 正文 | 14px - 16px | **15px** (PC) / **14px** (Mobile) |
| 辅助文字 | 12px - 14px | **13px** (通用) |

#### 建议方案
建立统一的字体层级系统：

```css
/* 字体大小规范 */
--text-4xl: 36px;  /* 超大标题，特殊场景 */
--text-3xl: 32px;  /* 页面大标题 */
--text-2xl: 24px;  /* 章节标题 */
--text-xl: 18px;   /* 卡片标题 */
--text-base: 15px; /* 正文 */
--text-sm: 13px;   /* 辅助文字 */
--text-xs: 12px;   /* 提示、标签 */
```

### 5. 阴影系统缺失 ⚠️ 轻微

#### 问题描述
阴影值随意设定，缺乏层级系统。

| 页面 | 阴影值 | 问题 |
|------|--------|------|
| 通知设置页 | `0 20px 60px rgba(0, 0, 0, 0.3)` | 过深 |
| 日历页 | `0 20px 60px rgba(0, 0, 0, 0.3)` | 过深 |
| 设置页 | `0 20px 60px rgba(0, 0, 0, 0.3)` | 过深 |
| 数据报表 | `0 2px 8px rgba(0,0,0,0.06)` | 过浅 |
| 徽章页 | `0 10px 30px rgba(243, 156, 18, 0.3)` | 特殊色 |

#### 建议方案
建立统一的阴影层级系统（采用紫色调，与品牌色一致）：

```css
/* 阴影规范 */
--shadow-sm: 0 2px 8px rgba(102, 126, 234, 0.08);      /* 轻微浮起 */
--shadow-md: 0 8px 24px rgba(102, 126, 234, 0.12);     /* 卡片默认 */
--shadow-lg: 0 16px 48px rgba(102, 126, 234, 0.16);    /* 弹窗、悬浮 */
--shadow-xl: 0 24px 64px rgba(102, 126, 234, 0.20);    /* 模态框 */
```

### 6. 间距系统不统一 ⚠️ 轻微

#### 问题描述
页面内边距、卡片间距、元素间距存在多种规格。

| 元素 | 发现的范围 | 推荐值 |
|------|-----------|--------|
| 页面内边距 (PC) | 30px - 40px | **40px** |
| 页面内边距 (Mobile) | 15px - 20px | **20px** |
| 卡片内边距 | 24px - 40px | **32px** |
| 卡片间距 | 20px - 30px | **24px** |
| 元素间距 | 12px - 20px | **16px** (基础单位) |

#### 建议方案
采用 8px 基准的间距系统：

```css
/* 间距规范 (8px 基准) */
--space-1: 8px;
--space-2: 16px;
--space-3: 24px;
--space-4: 32px;
--space-5: 40px;
--space-6: 48px;
--space-8: 64px;
```
