# 设置和隐私页面 E2E 测试用例文档

## 概述
本文档描述了设置页面和隐私页面的端到端(E2E)测试用例，使用Playwright框架实现。

## 测试范围

### 1. 设置页面测试 (settings.spec.ts)

#### 1.1 页面导航和显示测试
- **用例ID**: SET-001
- **标题**: 导航到设置页面并验证正确显示
- **前置条件**: 用户已登录
- **步骤**:
  1. 点击导航栏中的"设置"链接
  2. 验证URL是否包含/settings路径
  3. 验证页面标题包含"Settings"文本
- **预期结果**: 成功导航到设置页面且标题正确显示

#### 1.2 个人信息编辑测试
- **用例ID**: SET-002
- **标题**: 编辑用户个人资料信息
- **前置条件**: 用户已登录并访问设置页面
- **步骤**:
  1. 在显示名称输入框中填入新名称
  2. 在简介文本域中填入新信息
  3. 点击保存按钮
  4. 验证成功消息是否显示
  5. 验证更新后的信息是否正确显示
- **预期结果**: 个人信息成功更新并显示正确的成功消息

#### 1.3 密码更改测试
- **用例ID**: SET-003
- **标题**: 更改用户密码
- **前置条件**: 用户已登录并访问设置页面
- **步骤**:
  1. 点击"更改密码"标签页
  2. 输入当前密码
  3. 输入新密码
  4. 确认新密码
  5. 点击更新密码按钮
  6. 验证密码更改成功消息
- **预期结果**: 密码成功更改并显示成功消息

#### 1.4 通知偏好设置测试
- **用例ID**: SET-004
- **标题**: 更新通知偏好设置
- **前置条件**: 用户已登录并访问设置页面
- **步骤**:
  1. 点击"通知"标签页
  2. 切换邮件通知开关
  3. 切换推送通知开关
  4. 点击保存通知设置按钮
  5. 验证保存成功消息
- **预期结果**: 通知偏好设置成功保存并显示成功消息

#### 1.5 响应式布局测试
- **用例ID**: SET-005
- **标题**: 验证设置页面在不同屏幕尺寸下的响应式布局
- **前置条件**: 用户已登录并访问设置页面
- **步骤**:
  1. 设置移动设备视口(375x812)
  2. 验证移动菜单按钮是否可见
  3. 设置平板设备视口(768x1024)
  4. 验证页面内容正确加载
  5. 设置桌面设备视口(1280x720)
  6. 验证页面内容正确加载
- **预期结果**: 在所有屏幕尺寸下页面都正确显示和布局

### 2. 隐私页面测试 (privacy.spec.ts)

#### 2.1 页面导航和显示测试
- **用例ID**: PRIV-001
- **标题**: 导航到隐私页面并验证正确显示
- **前置条件**: 用户已登录
- **步骤**:
  1. 点击导航栏中的"隐私"链接
  2. 验证URL是否包含/privacy路径
  3. 验证页面标题包含"Privacy Settings"文本
- **预期结果**: 成功导航到隐私页面且标题正确显示

#### 2.2 隐私设置调整测试
- **用例ID**: PRIV-002
- **标题**: 调整用户隐私设置
- **前置条件**: 用户已登录并访问隐私页面
- **步骤**:
  1. 从个人资料可见性下拉菜单中选择"仅好友"
  2. 点击保存隐私设置
  3. 验证成功消息显示
  4. 从活动可见性下拉菜单中选择"私密"
  5. 点击保存隐私设置
  6. 验证成功消息显示
- **预期结果**: 隐私设置成功更新并显示成功消息

#### 2.3 数据共享偏好管理测试
- **用例ID**: PRIV-003
- **标题**: 管理数据共享偏好
- **前置条件**: 用户已登录并访问隐私页面
- **步骤**:
  1. 切换"与合作伙伴共享数据"选项
  2. 切换"允许分析追踪"选项
  3. 点击保存数据偏好
  4. 验证保存成功消息
- **预期结果**: 数据共享偏好成功保存并显示成功消息

#### 2.4 账号删除流程测试
- **用例ID**: PRIV-004
- **标题**: 执行账号删除流程
- **前置条件**: 用户已登录并访问隐私页面
- **步骤**:
  1. 滚动到账号删除部分
  2. 点击删除账号按钮
  3. 验证删除确认模态框出现
  4. 输入确认密码
  5. 点击最终删除按钮
  6. 验证重定向到确认页面或登录页面
- **预期结果**: 账号删除流程正确执行，用户被重定向

#### 2.5 账号停用流程测试
- **用例ID**: PRIV-005
- **标题**: 执行账号停用流程
- **前置条件**: 用户已登录并访问隐私页面
- **步骤**:
  1. 点击停用账号按钮
  2. 验证停用确认模态框出现
  3. 输入停用密码
  4. 点击确认停用按钮
  5. 验证停用成功消息
- **预期结果**: 账号停用流程正确执行，显示成功消息

#### 2.6 隐私页面响应式布局测试
- **用例ID**: PRIV-006
- **标题**: 验证隐私页面在不同屏幕尺寸下的响应式布局
- **前置条件**: 用户已登录并访问隐私页面
- **步骤**:
  1. 设置移动设备视口(375x812)
  2. 验证移动端隐私菜单是否可见
  3. 设置平板设备视口(768x1024)
  4. 验证页面内容正确加载
  5. 设置桌面设备视口(1280x720)
  6. 验证页面内容正确加载
- **预期结果**: 在所有屏幕尺寸下隐私页面都正确显示和布局

#### 2.7 隐私设置验证测试
- **用例ID**: PRIV-007
- **标题**: 验证隐私设置的正确保存和显示
- **前置条件**: 用户已登录并访问隐私页面
- **步骤**:
  1. 将个人资料可见性设置为"公开"
  2. 保存设置
  3. 重新加载页面
  4. 验证设置是否正确保留
- **预期结果**: 隐私设置在页面重载后仍保持正确值

## 测试环境要求
- Playwright v1.40+
- Node.js v16+
- 支持的浏览器(Chromium, Firefox, WebKit)

## 特殊注意事项
1. 所有测试用例都假设用户已登录系统
2. 测试中使用的元素选择器基于data-testid属性
3. 账号删除测试可能会影响真实数据，请在测试环境中运行
4. 响应式布局测试覆盖了常见的移动、平板和桌面设备尺寸
