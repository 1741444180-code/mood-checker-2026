# 性能测试用例文档

## 测试概述
本文档包含针对心情打卡网站的性能测试用例，涵盖页面加载、API 响应、用户体验等方面。

## 测试环境
- 浏览器: Chrome/Firefox/Safari
- 网络条件: 4G, WiFi
- 设备: 桌面端, 移动端
- 测试数据: 1000条心情记录

## 性能指标标准
- 首页加载时间: ≤ 3秒
- 页面切换时间: ≤ 2秒
- API响应时间: ≤ 2秒
- 表单提交响应: ≤ 3秒
- 图表渲染时间: ≤ 5秒
