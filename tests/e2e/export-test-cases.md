# 导出功能测试用例文档

## 测试概述
本文档描述了导出功能的端到端(E2E)测试用例，覆盖格式选择、日期范围选择、文件下载及响应式布局等关键功能。

## 测试环境
- 测试框架: Playwright
- 测试类型: E2E 测试
- 测试页面: /export

## 功能测试用例

### TC001: 页面元素可见性测试
**测试目的**: 验证导出页面加载时所有必要元素都可见
**前置条件**: 用户已导航至 /export 页面
**测试步骤**:
1. 打开浏览器并访问 /export 页面
2. 检查标题是否显示 "Export Data"
3. 检查导出格式选择器是否存在
4. 检查日期范围选择器是否存在
5. 检查导出按钮是否存在

**预期结果**: 所有上述元素都应在页面上可见

### TC002: 导出格式选择功能测试
**测试目的**: 验证用户可以选择不同的导出格式
**前置条件**: 用户已在导出页面
**测试步骤**:
1. 从下拉菜单中选择 "CSV" 格式
2. 验证选项已被选中
3. 选择 "Excel" 格式
4. 验证选项已被选中
5. 选择 "PDF" 格式
6. 验证选项已被选中

**预期结果**: 所有格式选项都可以成功选择

### TC003: 日期范围选择功能测试
**测试目的**: 验证用户可以设置日期范围进行数据导出
**前置条件**: 用户已在导出页面
**测试步骤**:
1. 在开始日期字段输入 "2023-01-01"
2. 验证开始日期字段值正确
3. 在结束日期字段输入 "2023-12-31"
4. 验证结束日期字段值正确

**预期结果**: 日期字段能正确接受并显示输入的日期值

### TC004: 文件下载功能测试
**测试目的**: 验证点击导出按钮后能够触发文件下载
**前置条件**: 用户已在导出页面并设置了导出参数
**测试步骤**:
1. 选择导出格式为 "CSV"
2. 设置开始日期为 "2023-01-01"，结束日期为 "2023-12-31"
3. 点击导出按钮
4. 等待文件下载完成
5. 验证下载的文件扩展名正确

**预期结果**: 文件成功下载且扩展名与所选格式匹配

### TC005: 响应式布局测试
**测试目的**: 验证导出页面在不同屏幕尺寸下的显示效果
**前置条件**: 用户已在导出页面
**测试步骤**:
1. 设置视口尺寸为桌面尺寸 (1920x1080)
2. 验证所有元素可见
3. 设置视口尺寸为平板尺寸 (768x1024)
4. 验证所有元素可见
5. 设置视口尺寸为手机尺寸 (375x667)
6. 验证所有元素可见

**预期结果**: 在所有屏幕尺寸下，页面元素都能正常显示

### TC006: 日期验证功能测试
**测试目的**: 验证系统能够正确处理无效的日期范围
**前置条件**: 用户已在导出页面
**测试步骤**:
1. 在开始日期字段输入 "2023-12-31"
2. 在结束日期字段输入 "2023-01-01"
3. 点击导出按钮
4. 检查错误消息是否显示

**预期结果**: 显示错误消息 "End date must be after start date"

### TC007: 选项持久性测试
**测试目的**: 验证页面刷新后用户的选择是否保持不变
**前置条件**: 用户已在导出页面
**测试步骤**:
1. 选择导出格式为 "Excel"
2. 设置开始日期为 "2023-06-01"，结束日期为 "2023-06-30"
3. 刷新页面
4. 验证格式选择仍为 "Excel"
5. 验证日期范围仍为设定值

**预期结果**: 刷新后用户的选择保持不变

## 性能指标
- 页面加载时间 < 3 秒
- 所有交互响应时间 < 1 秒
- 文件下载应在点击后 5 秒内开始

## 浏览器兼容性
- Chrome (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- Edge (最新版本)

## 测试数据
- 开始日期: 2023-01-01
- 结束日期: 2023-12-31
- 支持的格式: CSV, Excel, PDF
