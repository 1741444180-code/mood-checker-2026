# 徽章和排行榜E2E测试用例文档

## 1. 测试概述

### 1.1 测试目标
- 验证徽章页面的功能完整性和用户体验
- 验证排行榜页面的数据准确性和交互功能
- 确保响应式布局在不同设备上正常显示
- 测试徽章解锁机制的正确性

### 1.2 测试范围
- 徽章页面的所有功能模块
- 排行榜页面的数据展示和交互
- 移动端、平板端、桌面端的响应式适配
- 用户界面的可用性和易用性

## 5. 测试执行说明

### 5.1 环境要求
- Playwright 测试框架
- Node.js 16+
- 支持的浏览器（Chromium, Firefox, WebKit）

### 5.2 执行命令
```bash
# 运行所有测试
npm run test:e2e

# 运行徽章测试
npx playwright test tests/e2e/badges.spec.ts

# 运行排行榜测试
npx playwright test tests/e2e/leaderboard.spec.ts

# 运行测试并打开UI模式
npx playwright test --ui
```

### 5.3 报告生成
测试执行后会自动生成HTML报告，位于 `playwright-report/` 目录下。

## 6. 预期输出

### 6.1 成功指标
- 所有测试用例通过率 100%
- 页面加载时间 < 3秒
- 响应式布局在各设备上正常显示
- 数据准确性验证通过

### 6.2 失败处理
- 如果测试失败，检查网络连接
- 确认测试环境中的数据完整性
- 验证页面元素的选择器是否正确
