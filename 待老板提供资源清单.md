1. 登录 Vercel Dashboard
2. 进入你的项目（或创建新项目）
3. 点击 "Storage" 标签
4. 点击 "Add Database" → "Create new database"
5. 选择 "Postgres"
6. 填写数据库名称：`mood-checker-db`
7. 选择区域：推荐 `Washington, D.C. (iad1)` (离中国较近)

2. 选择你的项目
3. 复制环境变量：
   - `POSTGRES_URL`

## 五、常见问题

### Q1: 为什么要 GitHub？
A: GitHub 是代码托管平台，支持版本管理、协作开发，Vercel 可以直接从 GitHub 自动部署。

### Q2: 为什么要 Vercel？
A: Vercel 是免费的部署平台，支持 Next.js 自动部署，全球 CDN 加速，自动 HTTPS，非常适合这个项目。

### Q3: 数据库可以用其他的吗？
A: 可以，但 Vercel Postgres 最简单（一键创建，自动配置），免费额度够用。其他选项包括：
- Railway (免费额度)
- Supabase (免费额度)
- 自建 PostgreSQL

### Q4: 部署后访问地址是什么？
A: Vercel 会自动分配一个免费域名：
```
https://mood-checker-xxx.vercel.app
```
也可以绑定自己的域名（可选）。

### Q5: 免费额度够用吗？
A: 完全够用！
- Vercel: 免费 100GB 流量/月
- Vercel Postgres: 免费 1GB 存储
- GitHub: 免费无限公开/私有仓库
