## 🛠️ 测试工具

### 自动化测试脚本（小林准备）

**创建测试数据脚本：**
```bash
# scripts/seed-mood-data.js
# 用法：
node scripts/seed-mood-data.js --count 100    # 生成 100 条
node scripts/seed-mood-data.js --count 1000   # 生成 1000 条
node scripts/seed-mood-data.js --reset        # 清空测试数据
```

**批量测试脚本：**
```bash
# scripts/run-all-tests.sh
npm run test:e2e          # E2E 测试
npm run test:performance  # 性能测试
npm run test:load         # 负载测试
```

### 浏览器开发者工具（小陈使用）

**检查点击无响应：**
1. 打开 Chrome DevTools (F12)
2. 切换到 Console 标签
3. 点击按钮，查看是否有报错
4. 切换到 Network 标签，查看请求是否发送

**检查性能问题：**
1. 打开 Performance 标签
2. 点击录制
3. 操作页面
4. 停止录制，分析瓶颈
