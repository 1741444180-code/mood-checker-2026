## 五、测试环境配置详情

### 5.1 工具配置
- **Playwright**: 用于E2E测试，支持多浏览器
- **Jest**: 用于单元测试和API测试
- **测试框架**: TypeScript支持，完整的测试生命周期

### 5.2 测试目录结构
```
tests/
├── e2e/                    # 端到端测试
│   └── mood-checker.e2e.spec.ts
├── unit/                   # 单元测试 (待完善)
└── api/                    # API测试 (待完善)
```

### 5.3 可用测试命令
- `npm run test` - 运行Jest单元测试
- `npm run test:e2e` - 运行Playwright E2E测试
- `npm run test:unit` - 运行单元测试
- `npm run test:api` - 运行API测试
