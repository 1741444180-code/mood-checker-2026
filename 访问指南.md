# 心情打卡网站 - 访问指南

**本地访问地址：** http://localhost:3000

---

## 🔧 如果看到空白页

### 解决方案 1：强制刷新浏览器

**Windows/Linux:**
- Ctrl + Shift + R
- 或 Ctrl + F5

**Mac:**
- Cmd + Shift + R

---

### 解决方案 2：清除浏览器缓存

**Chrome:**
1. 按 F12 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

**Safari:**
1. 开发 > 清空缓存
2. 历史记录 > 清除历史记录

---

### 解决方案 3：检查服务器状态

打开终端执行：
```bash
curl -I http://localhost:3000
```

应该返回：
```
HTTP/1.1 200 OK
```

---

### 解决方案 4：重启服务器

```bash
# 停止服务器
pkill -f "next dev"

# 重启服务器
cd /Users/lijianquan/.openclaw/workspace/projects/mood-checker/frontend
npm run dev
```
