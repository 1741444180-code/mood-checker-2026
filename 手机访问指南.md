## 🚀 方案 A：部署到 Vercel（推荐）

### 优势
- ✅ 免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 一键部署
- ✅ 自动生成公网 URL

### 部署步骤

**1. 访问 Vercel**
```
https://vercel.com
```

**2. 登录 GitHub 账号**
- 使用 GitHub 账号登录
- 授权 Vercel 访问仓库

**3. 导入项目**
- 点击 "Add New Project"
- 选择 `mood-checker-2026` 仓库
- 点击 "Import"

**4. 配置构建设置**
- Framework Preset: Next.js
- Root Directory: `frontend`
- Build Command: `npm run build`
- Output Directory: `.next`

**5. 点击 "Deploy"**
- 等待 2-3 分钟
- 部署成功后获得公网 URL

**6. 获取访问地址**
```
https://mood-checker-2026.vercel.app
```

### 手机访问

**方式 1：直接访问**
```
在手機瀏覽器輸入：https://mood-checker-2026.vercel.app
```

**方式 2：扫描二维码**
- Vercel 部署成功后会显示二维码
- 用手机扫描二维码直接访问

## 🌐 方案 B：使用内网穿透工具

### ngrok（推荐）

**1. 安装 ngrok**
```bash
brew install ngrok
```

**2. 启动 ngrok**
```bash
ngrok http 3000
```

**3. 获取公网 URL**
```
Forwarding: https://xxx-xxx-xxx.ngrok.io -> http://localhost:3000
```

**4. 手机访问**
```
在手機瀏覽器輸入：https://xxx-xxx-xxx.ngrok.io
```

**注意：** ngrok 免费版 URL 每次重启会变

## 📱 方案 C：使用局域网 + 公网 IP

### 前提条件
- 家里有公网 IP
- 路由器已配置端口转发

### 配置步骤

**1. 查看本机 IP**
```bash
ifconfig | grep "inet "
```

**2. 配置路由器端口转发**
- 登录路由器管理页面
- 添加端口转发规则
- 外部端口：3000
- 内部 IP：[你的电脑 IP]
- 内部端口：3000

**3. 查看公网 IP**
```bash
curl ifconfig.me
```

**4. 手机访问**
```
在手機瀏覽器輸入：http://[你的公网IP]:3000
```

**注意：** 需要电脑保持开机，且服务器运行中

## 🔧 常见问题

### Q1: Vercel 部署失败？
**解决：** 检查 `frontend` 目录是否有 `package.json`

### Q2: 手机访问显示空白？
**解决：** 强制刷新浏览器（Cmd+Shift+R）

### Q3: 样式不正常？
**解决：** 清除浏览器缓存

### Q4: API 无法访问？
**解决：** 检查环境变量配置
