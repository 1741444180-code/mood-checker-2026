### 11.3 咪咪 - 后端开发

| 项目 | 内容 |
|------|------|
| **任务** | 后台管理 API 开发 |
| **预计完成** | 3 小时 |
| **交付物** | 后台管理 API 接口 |
| **交付位置** | `src/app/api/admin/` |

**具体要求：**
1. `GET /api/admin/users` - 用户列表（支持分页、搜索）
2. `PUT /api/admin/users/:id/status` - 修改用户状态
3. `GET /api/admin/content` - 内容列表（心情/评论）
4. `DELETE /api/admin/content/:id` - 删除内容
5. `GET /api/admin/stats` - 统计数据
6. `POST /api/admin/announcement` - 发布公告

**技术栈：** Next.js API Routes + Prisma + PostgreSQL
